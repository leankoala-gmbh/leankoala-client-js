# Testing the KoalityEngine client

For unit testing we are using the JEST library. 

* [The JEST library](https://jestjs.io/)

## Running the tests

```shell script
npm run test
```

## Code coverage

To see the code coverage run
```shell script
node node_modules/jest/bin/jest.js --collectCoverage
```
